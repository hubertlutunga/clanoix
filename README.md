# clanoix

Site vitrine de la maison d’édition CLANOIX, présentant le livre *Naviguer dans la prise de poste* et les prochaines publications.

## Configuration

Copier `config.example.php` vers `config.php`, puis renseigner les accès MySQL et SMTP.

`config.php` est ignoré par Git afin de ne pas publier les identifiants sensibles.
